## Test_IDF_Procurement_API_-NestJS-simple-app
this project is build a basic manage of buing wepons, you can buy a wepon ens store all of the data on the wepon in db

### install:
```
nest new simple-app
npm i class-validator class-transformer
npm install --save @nestjs/typeorm typeorm mysql2
npm install --save @nestjs/sequelize sequelize sequelize-typescript mysql2
npm install --save-dev @types/sequelize
```
### run the project: 
```
npm run start:dev
```
